Able Video Player

Audio description via VTT track, read by browsers (Uses Speech synthesis Interface)

Able Player can use the Speech Synthesis Interface to read audio descriptions directly from a WebVTT (.vtt) file. This eliminates the need for a separate .mp3 file for audio descriptions or a secondary .mp4 video with embedded narration. However, this feature has certain limitations and works only in browsers that support the Speech Synthesis API.

Audio description via video swap

This demo uses a separate video to deliver audio description. If Description is toggled on, the user gets the described version; otherwise they get the non-described version.

Extended Audio description via video swap

This demo uses a separate video to deliver audio description. If Description is toggled on, the user gets the described version; otherwise they get the non-described version.

In the described version, the video contains built-in pauses that occur whenever the audio description is read aloud. This results in a described version and non-described version that have different durations.
